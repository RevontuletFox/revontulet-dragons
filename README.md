# 龍學 · A Compendium of Dragons

A six-volume meditation on dragons — written in HTML, read in browsers, addressed to anyone who has felt that "the dragon" is more than a symbol.

> 被讲述的龙，会死。<br>
> 不讲述自己的龙，不死。
>
> *Dragons that get told die. Dragons that don't tell themselves don't.*

## 这是什么 / What this is

六篇长文，一个共同的论题——*龙不是被发现的，是被需要的*。

Six long-form essays, one shared thesis — *the dragon was never discovered, it was needed.*

每篇都是一个完整的 HTML 页面，自带独特的色温与排版美学。可以按编号顺序读，也可以从任何一篇切入。

Each essay is a self-contained HTML page with its own typographic temperament. Read in order, or start anywhere that calls to you.

## 六卷 / The Six Volumes

### 卷壹 · *看见* / Volume I · *Seeing*
**龙之百科 · A Compendium · From Myth to Self**

世界九种龙文化的并置巡礼，落点在一句本体论命题：龙是不依赖外界确认而存在的。

A panoramic survey of nine dragon traditions, landing on a single ontological claim: a dragon does not require external confirmation to exist.

→ [阅读卷壹 / Read Volume I](https://revontuletfox.github.io/revontulet-dragons/01-龙之百科-compendium.html)

---

### 卷貳 · *理解* / Volume II · *Understanding*
**一个符号的多重宇宙 · Eleven Lenses · The Meaning of the Dragon**

十一道学科之光照向同一条中国龙——考古、文字、神话、哲学、天文、政治、艺术、民俗、比较文化、深层心理、符号经济。

Eleven disciplinary lenses refracted through one Chinese dragon — archaeology, paleography, mythology, philosophy, astronomy, politics, art, folklore, comparative culture, depth psychology, semiotic economy.

→ [阅读卷貳 / Read Volume II](https://revontuletfox.github.io/revontulet-dragons/02-龙的意义-meaning.html)

---

### 卷叁 · *守护* / Volume III · *Defending*
**龙的死亡与重生 · Death &amp; Rebirth · After Reason, Capital and AI**

龙在四种死法（理性、政治、消费、AI 生成）之后，由三条路径复活——孩子、反叙事、自我宣称。

After four deaths in modernity (reason, politics, consumption, AI generation), the dragon returns through three pathways — the child's imagination, counter-myth, and self-claim.

→ [阅读卷叁 / Read Volume III](https://revontuletfox.github.io/revontulet-dragons/03-龙的死亡与重生-death-rebirth.html)

---

### 长文 · *承认* / Long Essay · *Acknowledging*
**为什么人类一直需要龙 · Why Humans Have Always Needed Dragons**

七位思想家（Vico / Eliade / Lévi-Strauss / Jung / Tolkien / Benjamin × Lacan / Heidegger）的七种回答。末尾有一间"私语小房间"——你可以写下自己的回答，仅存浏览器本地。

Seven thinkers, seven answers. A private chamber at the end where you write your own; the text stays only in your browser.

→ [阅读长文 / Read the Long Essay](https://revontuletfox.github.io/revontulet-dragons/04-为什么人类一直需要龙-why-we-need-dragons.html)

### 卷伍 · *化身* / Volume V · *Becoming*
**化龍 · The Becoming · From acknowledging to becoming**

乾卦六龙位（潜 / 见 / 惕 / 跃 / 飞 / 亢）与当代化龙的三条窄路——创作、孤独、传递。

The six stations of the Qian hexagram, and three modern paths into the becoming — making, solitude, transmission.

→ [阅读卷伍 / Read Volume V](https://revontuletfox.github.io/revontulet-dragons/05-化龍-becoming.html)

### 卷陸 · *聆听* / Volume VI · *Listening*
**龍的對話錄 · A Symposium · One night at the brazier**

仿柏拉图体。七位思想家与你一夜对谈，外加一位整夜不说话的衔尾蛇——它在最后一幕里做了一件事，只有留下来的人能听见。

A Platonic-style symposium. Seven thinkers in dialogue with you — and one silent ouroboros who breathes the whole night through. What it "says" can only be heard by whoever stays until the end.

## 閱讀路徑 / Reading Paths

* **顺序读 · In order**：壹 → 貳 → 叁 → 长文 → 伍 → 陸
  Sequential — recommended for first encounter.

* **快路 · The short path**：先读 [长文 / the long essay](https://revontuletfox.github.io/revontulet-dragons/04-为什么人类一直需要龙-why-we-need-dragons.html)（七答 / seven answers），再读 [卷陸 / Volume VI](https://revontuletfox.github.io/revontulet-dragons/06-龍的對話錄-symposium.html)（七夜对话 / seven nightly dialogues）。
  Quick path — two essays that compress the whole arc.

* **自助 · Pick at random**：每一篇都自带独立的视觉与论题，从任何一篇切入都不丢上下文。
  Each essay is self-standing — entering from any chapter is fine.

## 技术说明 / Notes on the Build

* 纯 HTML / CSS / 极简 JS — 无构建工具，无框架。
  Plain HTML / CSS / minimal JS — no build tools, no framework.

* 字体均自 Google Fonts：Cinzel · Cormorant Garamond · EB Garamond · Noto Serif SC · Ma Shan Zheng · ZCOOL XiaoWei。
  All typefaces from Google Fonts.

* 每卷独立色温档：夜色暗金 · 日间米黄 · 暮色余烬 · 黎明紫金 · 翠玉破晓 · 炉火夜话。
  Each volume keeps its own colour temperature — night-gold, day-cream, dusk-ember, dawn-violet, jade-morning, brazier-night.

* 通过 GitHub Pages 部署，纯静态托管。
  Hosted as a static site via GitHub Pages.
